# Python Regex Operations

This folder contains Python scripts demonstrating common regular expression operations using the `re` module.

| File | Description |
|------|-------------|
| re.search.py | Basic `re.search()` usage to find a pattern in a string |
| regex-findall.py | Using `re.findall()` to find all matches in a string |
| regex-match.py | Using `re.match()` to match patterns at the start of a string |
| regex-replace.py | Using `re.sub()` to replace patterns in a string |
| regex-search.py | Simple chatbot using `re.search()` with `re.IGNORECASE` for case-insensitive matching |
| regex-split.py | Splitting strings using regex patterns with `re.split()` |
