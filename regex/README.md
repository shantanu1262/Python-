# Python Regex Operations

This folder contains Python scripts demonstrating common regular expression operations using the `re` module.

| File | Description |
|------|-------------|
| re.search.py | Using `re.search()` to find a pattern in a string |
| regex-findall.py | Using `re.findall()` to find all matches |
| regex-match.py | Using `re.match()` to match patterns at the start |
| regex-replace.py | Using `re.sub()` to replace patterns |
| regex-search.py | Searching for patterns in strings |
| regex-split.py | Splitting strings using regex patterns |
