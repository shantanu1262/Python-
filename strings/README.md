# Python String Operations

This folder contains Python scripts demonstrating common string operations.

| File | Description |
|------|-------------|
| String-concat.py | String concatenation examples |
| string-len.py | Getting the length of a string |
| string-lowercase.py | Converting strings to lowercase |
| string-replace.py | Replacing substrings |
| string-split.py | Splitting strings |
| string-strip.py | Stripping whitespace from strings |
| string-substring.py | Extracting substrings |
